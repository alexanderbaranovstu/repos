Модуль который мы будем испорльзовать для сбора общих для группы аварий
нам необходимо добиться что бы если хотя бы один элемент в групе находится в аварии
то вся группа отображает состояние аварии
если аврий нет, и хотя бы один элемент в группе находится в работе - то показываем работу
если все элементы в группе выключены - то и группа выключена
элемент группы переведенный в состояние сервис - не обрабатывается
Базовые состояния по которым происходит переключение алгоритмов работы следующие
  enum BUState
  {
    range
    {
      off,
      standby,
      on,
      warning,
      alarm,
      service
    }
  }
Проблема в том что по разным причинам это состояние unit не всегда именно в таком виде поступает из поля
по этому необходимо иметь возможность преобразовать состояние полевого устройство в базовое

для этого придуман класс BComponentsToAction позволяющий вызвать нужный метод при наличии условия в маске
    properties  
    {  
       in           :     BString     flags { summary }      default {[ "" ]}     
       mask         :     BString     flags { summary }      default {[ "" ]}     
       parentAction :     BString     flags { summary }      default {[ "" ]}     
    }   
таким образом если на вход попадает значение соответствующее маске - для родителя вызывается метод указанный в parentAction
